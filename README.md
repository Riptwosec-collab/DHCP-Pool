# 🌐 DHCP Automation PRO

เครื่องมือสร้าง DHCP Pool สำหรับ Cisco IOS - อัตโนมัติและใช้งานง่าย

![DHCP Tool](https://img.shields.io/badge/Version-2.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ คุณสมบัติเด่น

- **MAC Auto Format** → แปลงเป็นรูปแบบ `0100.1500.2560.aa` อัตโนมัติ
- **Bulk Import** → นำเข้าหลาย Pool พร้อมกัน
- **Auto Fill IP** → เติม IP แบบต่อเนื่องอัตโนมัติ
- **Validation แบบ Real-time** → แสดงสีแดงเมื่อข้อมูลไม่ครบ
- **ปรับแต่ง DNS Server** ได้
- **เปิด/ปิด ip source binding** ได้
- **Export เป็นไฟล์ .cfg**

## 🚀 วิธีใช้งาน

1. ดาวน์โหลดไฟล์ `dhcp-automation.html`
2. ดับเบิลคลิกเปิดด้วยเบราว์เซอร์ (Chrome / Edge แนะนำ)
3. เริ่มกรอกข้อมูลหรือใช้ **Mass Pool Import**

## 🛠️ คุณสมบัติหลัก

- **MAC Format** อัตโนมัติ (รองรับหลายรูปแบบ)
